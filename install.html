Installation Instructions for KarinaCraft

I'm quite lazy, so this is just going to be basic html. Follow for step-by-step instructions (Windows only)

1. Download the Technic Launcher here

2. Install by running the downloaded .exe file.

3. Open the newly installed Technic Launcher.

4. Log in with your Minecraft credentials.

5. Click on Modpacks in the upper nav-bar.

6. Search KarinaCraft in the search bar.

7. Click "Install"

8. Once installed, click "Play"

8.1 If java not installed, click here

9. After launch, navigate to "Multiplayer" and enter "karinacraft.anadyr.us"

10. Profit!
